# 圖片文件放置說明

此目錄用於存放「碳智匯」系統的品牌圖片文件。

## 📸 需要的圖片文件

請將以下圖片文件放置在此目錄中：

### 1. logo.png
- **用途**：系統 Logo
- **建議尺寸**：512 x 512 像素
- **格式**：PNG（建議使用透明背景）
- **顯示位置**：
  - 導航欄左上角
  - 側邊欄品牌區
  - 首頁主要展示區域

### 2. background-horizontal.png
- **用途**：桌面端背景圖
- **建議尺寸**：1920 x 1080 像素（或更大）
- **格式**：PNG 或 JPG
- **顯示位置**：
  - 所有頁面的背景（桌面端）
  - 固定定位，提供視差效果

### 3. background-vertical.png
- **用途**：移動端背景圖
- **建議尺寸**：1080 x 1920 像素（或更大）
- **格式**：PNG 或 JPG
- **顯示位置**：
  - 所有頁面的背景（移動端）

## 📁 完整文件結構

放置圖片後，此目錄應包含：

```
public/images/
├── README.md                      (本文件)
├── logo.png                       (系統 Logo)
├── background-horizontal.png      (桌面背景)
└── background-vertical.png        (移動端背景)
```

## 🎨 設計建議

### Logo (logo.png)
- 使用圓形設計或在透明背景上的圓形元素
- 包含代表「智慧」和「環保」的視覺元素
- 確保在小尺寸下仍清晰可辨
- 建議使用藍色和綠色漸變配色

### 背景圖片
- 使用科技感電路板紋理
- 結合綠色葉子或自然元素
- 保持柔和的色調以確保前景內容可讀
- 建議色系：淡藍色、淡綠色、白色
- 避免過於複雜的圖案

## ⚙️ 技術說明

### 背景圖片的應用
系統使用 CSS 類來應用背景圖片：

- **桌面端**：`.bg-pattern` 類使用 `background-horizontal.png`
- **移動端**：`.bg-pattern-vertical` 類使用 `background-vertical.png`
- **覆蓋層**：`.bg-overlay` 類添加半透明白色漸變覆蓋層，確保內容可讀性

### 自動優化
- Next.js Image 組件自動優化 Logo 圖片
- 支援響應式加載，根據設備屏幕大小提供適當尺寸
- 自動處理圖片懶加載

## 🔧 測試圖片

放置圖片後，請執行以下步驟測試：

1. **重啟開發服務器**：
   ```bash
   npm run dev
   ```

2. **訪問以下頁面檢查圖片顯示**：
   - 首頁：http://localhost:3000
   - Dashboard：http://localhost:3000/dashboard
   - 其他頁面

3. **檢查項目**：
   - ✓ Logo 在導航欄正確顯示
   - ✓ 首頁大 Logo 正確顯示
   - ✓ 側邊欄 Logo 正確顯示
   - ✓ 背景圖在桌面端正確顯示
   - ✓ 背景圖在移動端正確顯示（使用瀏覽器開發者工具測試）
   - ✓ 所有文字內容清晰可讀

## ⚠️ 注意事項

1. **文件名必須完全匹配**（包括大小寫和副檔名）
2. **建議優化圖片大小**以提升載入速度：
   - Logo：< 100KB
   - 背景圖：< 500KB
3. **使用 PNG 格式**以獲得最佳質量
4. **測試不同屏幕尺寸**確保圖片在各種設備上都顯示正常
5. **提交圖片到 Git**：
   ```bash
   git add public/images/
   git commit -m "feat: 添加品牌圖片資源"
   ```

## 🆘 常見問題

### Q: 圖片沒有顯示怎麼辦？
A: 請檢查：
- 文件名是否完全匹配（包括副檔名）
- 圖片是否在正確的目錄中
- 是否已重啟開發服務器
- 瀏覽器控制台是否有錯誤訊息

### Q: 背景圖太暗，影響內容閱讀？
A: 系統已添加半透明覆蓋層。如需調整，可修改 `globals.css` 中 `.bg-overlay::before` 的 `background` 屬性。

### Q: 需要不同尺寸的 Logo 嗎？
A: 不需要。系統會自動根據顯示位置調整 Logo 大小。只需提供一個高解析度的原始 Logo 即可。
